Instagreen


Instagreen is a web application that encourages users to take a resolution monthly/weakly/daily to contribute in saving earth.
In this web app we have provided a dedicated page to take resolutions, and a page for calculating carbon footprints using a carbon footprint calculator embedded in it.
Also the users can view the activity on GreenFeed(static for now)
